para rodar o sistema é necessario fazer mudanças de conta e senha no application.properties no back end

apos isso, sera necessario rodar o ReservaPousadasAplication.java

agora pro frontend apos a instação você fara usara o comando cd pousadas-frontend e depois o comando npm install

apos isso vc pode rodar o sistema usando o comando ng serve.
